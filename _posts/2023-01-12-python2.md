# boolean, 논리연산자

## 1. 객체가 같은지 다른지 비교

==, != 는 값 자체를 비교하고,

is, is not 은 객체를 비교한다. 

**소스**

```python
print(1 == 1.0)
print(1 is 1)
print(1 is not 1.0)
print(1 is 1.0) # 1은 정수객체, 1.0은 실수객체
```
**결과**

```plaintext
True
True
True
False
```
